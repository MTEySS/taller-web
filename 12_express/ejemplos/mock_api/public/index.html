Mock-api tests

query records

  • get all records: /api/usuarios
  • get all records order by name: /api/usuarios?order=nombre
  • get all records order by name descending: /api/usuarios?order=nombre&sort=desc
  • get first 3 records: /api/usuarios?len=3
  • get second page of 3 records: /api/usuarios?page=2&len=3
  • get second page order by age: /api/usuarios?order=age&page=2&len=3
  • get records containing ma: /api/usuarios?q=ma
  • get records containing ma in the field nombre: /api/usuarios?q=ma&fields=nombre

query a specific record

  • get record with id 2: /api/usuarios/2
  • get a non existent record: /api/usuarios/999
