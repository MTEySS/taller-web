TDD demo

Esta es una aplicación de ejemplo para mostrar cómo se podría utilizar la función cuit.esValido en un alta de un registro. Al tipear cualquier caracter en el input box de cuit, se ejecutará la función de validación. Si no hay un cuit válido, le agregará el class has-error para indicar que el valor del campo es erróneo.

Inicialmente la funcion cuit.esValido sólo retorna verdadero cuando ingresa un '1'. Puede probarlo ingresando distintos valores en el input box. El ejercicio consiste en agregar pruebas unitarias en test/spec/cuitSpec.js y luego implementar correctamente la función cuit.esValido en cuit.js.

Alta de usuario

Debe ingresar un número de cuit válido.
